Read Me file for the "Bada-Boom!!" Project.

Game Created by Luis Rangel and Alfredo Deleon
Original Engine provided by Dr. Ian Parberry
Art and Sounds credited in the coding comments

---Key Bindings---
ARROW KEYS-	Player walks using the arrow keys.
SPACE-		Punch
ENTER-		Start Game(On Startscreen/Game Over Screen)
			Pause/Unpause(Elsewhere)
ESC-			Quit

---Game Objective---
Surivive as long as you can and get a high score by defeating enemies.
